# MongoDB

## 🛠 Installation

### Linux + WSL2

> Note: The following script is only supported on Ubuntu

```bash
chmod +x install_mongodb.sh
sudo ./install_mongodb.sh
```

### Windows

> Note: When prompted by the installer choose to install MongoDB as a service

```bash
.\install_mongodb.bat
```
